# Товары: один товар — одна папка

Правила:

- **Один товар = одна папка** с именем = ID товара (латиница, цифры, дефис, без пробелов). Примеры: `poster-1`, `azbuka`, `tabela`.
- Внутри папки — один файл **product.json** с данными этого товара.
- Список всех товаров и их порядок в каталоге — в **index.json** в этой папке.

Папка **example** — образец. Для нового товара копируйте её, переименуйте в свой ID, отредактируйте product.json и добавьте ID в index.json.

Картинки этого товара лежат в **site/images/products/<тот-же-id>/** — так данные и картинки одного товара не пересекаются с другими.
