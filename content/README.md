# Контент: данные сайта

## Товары

Все товары лежат в **products/** по правилу «один товар — одна папка»:

- **products/index.json** — список ID товаров (порядок вывода в каталоге).
- **products/<id>/product.json** — данные одного товара. ID = имя папки.
- **products/example/** — образец. Для нового товара: новая папка с тем же ID, что и в **site/images/products/<id>/** для картинок.

Подробные правила — в **docs/FILE_STRUCTURE.md**.

Картинки товаров — в **site/images/products/<id>/** (не здесь). Так данные и картинки одного товара не пересекаются с другими.
