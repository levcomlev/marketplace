# Контент: товары и данные

Здесь хранится структура данных для товаров (плакатов). Используйте как образец.

- **products.example.json** — пример полей одного товара и списка.
- Когда будете подставлять свои товары, можно скопировать файл в `products.json` (и добавить `content/products.json` в `.gitignore`, если там будут только данные без секретов) или вести контент в своей CMS.

В вёрстке каталог и карточку товара можно сначала сверстать на статичном HTML, потом подключить подстановку из JSON или из API.
