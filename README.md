# Маркетплейс-витрина: обучающие плакаты

Сайт-витрина товаров (обучающие плакаты): реклама ведёт сюда, пользователь выбирает — купить на Озоне, Wildberries или цифровую версию с сайта.

## Что есть в проекте

- **PLAN.md** — план разработки, этапы, ваш контекст (плакаты, цифра, ИП).
- **docs/WORKFLOW.md** — как работать: маленькие шаги, коммит после каждой задачи, как откатиться.
- **docs/PAYMENT_TAX.md** — оплата и чеки по 54-ФЗ для ИП (чтобы не беспокоила налоговая).
- **docs/SITE_STRUCTURE.md** — список страниц и блоков для вёрстки по шагам.
- **docs/BACKLOG.md** — чек-листы: доступность, мета/SEO, аналитика, 404, подписка и т.д.; порядок внедрения.
- **design/design-tokens.md** — цвета, шрифты, отступы (один источник правды для стилей).
- **content/** — данные товаров: один товар = одна папка в `content/products/<id>/`, список в `index.json` (см. **docs/FILE_STRUCTURE.md**).
- **site/** — исходники сайта: HTML, CSS, JS, изображения. Вёрстка добавляется постепенно.

## Как работать

1. **Один логический шаг — один коммит.** Так проще откатиться (см. docs/WORKFLOW.md).
2. Верстать в порядке: главная → каталог → карточка товара (см. docs/SITE_STRUCTURE.md).
3. Стили и контент — в отдельных файлах; общие величины брать из design/design-tokens.md.
