# Исходники сайта (вёрстка)

Здесь живёт вёрстка: HTML, CSS, JS, изображения.

- **index.html** — главная страница (пока заглушка).
- **css/** — стили. Рекомендуется: один файл с токенами (`tokens.css`), отдельные файлы под страницы или блоки (например `header.css`, `catalog.css`).
- **js/** — скрипты. Минимум на старте; позже — корзина, оплата, подстановка данных из `content/`.
- **images/** — картинки: логотипы, фото плакатов, иконки.

Вёрстайте по шагам (см. `docs/SITE_STRUCTURE.md`), каждый шаг — коммит.
