# Изображения

- **shared/** — общие: логотип бренда, иконки (Озон, Wildberries), декор. Не класть сюда картинки товаров.
- **products/<id>/** — только картинки одного товара; ID совпадает с папкой в `content/products/<id>/`. Одна папка на товар — без пересечений.

Подробные правила — **docs/FILE_STRUCTURE.md**. Рекомендуется оптимизировать размер и формат (WebP где возможно) для рекламного трафика.
