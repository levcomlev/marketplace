# Картинки товаров: одна папка на товар

Правило: **один товар = одна папка**. Имя папки = ID товара (как в `content/products/<id>/`).

Примеры:
- `site/images/products/guitar/` — картинки товара `guitar`
- `site/images/products/yoga_cards/` — картинки товара `yoga_cards`

## Именование: 1, 2, 3…

Кладите фото в папку товара с **номерами**: `1.jpg`, `2.jpg`, `3.jpg` (подойдут и `.png`, `.webp`).

- **1** — главное фото (им пользуются каталог и галерея; оно же превью).
- **2, 3, 4…** — остальные кадры галереи на странице товара (порядок по номеру).

Папка: `site/images/products/<id>/`, например `site/images/products/guitar/` с файлами `1.jpg`, `2.jpg`, `3.jpg`.

Скрипт сборки сам найдёт такие файлы и соберёт галерею. Вручную прописывать пути в `product.json` не нужно.

После добавления или смены фото выполните из корня проекта:  
`node scripts/build-products-json.js`
