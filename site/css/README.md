# Стили

Один блок/функция = один файл (подробно — **docs/FILE_STRUCTURE.md**).

- **tokens.css** — переменные (цвета, шрифты, отступы).
- **base.css** — сброс, типографика, общие стили страницы.
- **header.css**, **footer.css** — шапка и подвал.
- **home.css** — главная (герой, «Где купить», хиты).
- **catalog.css**, **product.css** — каталог и карточка товара.

Подключать в HTML в порядке: tokens → base → нужные блоки для страницы.
