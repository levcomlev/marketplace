# Стили

Рекомендуемая структура:

- **tokens.css** — переменные из `design/design-tokens.md` (цвета, шрифты, отступы).
- **base.css** или **global.css** — сброс, типографика, общие стили страницы.
- **header.css**, **footer.css** — шапка и подвал.
- **catalog.css**, **product.css** — стили каталога и карточки товара.

Подключать в HTML в нужном порядке: сначала tokens, потом base, потом блоки.
