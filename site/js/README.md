# Скрипты

На этапе витрины (без оплаты) можно обойтись минимумом:

- Подключение данных из `content/products.json` для каталога и карточки товара (по желанию).
- Мобильное меню (открытие/закрытие).
- Плавная прокрутка или другие мелкие улучшения.

Файлы лучше держать отдельно (например `menu.js`, `catalog.js`), подключать по мере необходимости. Каждое логическое добавление — отдельный коммит.
