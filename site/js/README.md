# Скрипты

Одна функция = один файл (подробно — **docs/FILE_STRUCTURE.md**).

- **menu.js** — мобильное меню (открытие/закрытие).
- **catalog.js** — каталог (подстановка карточек из данных по желанию).
- **product.js** — страница товара (подстановка данных по ID по желанию).

Подключать только нужные для страницы. Не смешивать логику разных блоков в одном файле.
