# Дизайн-токены

Один файл с базовыми величинами: цвета, шрифты, отступы. При вёрстке используйте эти значения, чтобы стили были едиными и их можно было поменять в одном месте.

## Цвета (пример — подставьте свои)

| Название | Назначение | Пример значения |
|----------|------------|------------------|
| `--color-bg` | Фон страницы | `#ffffff` |
| `--color-text` | Основной текст | `#1a1a1a` |
| `--color-text-muted` | Второстепенный текст | `#666` |
| `--color-accent` | Кнопки, ссылки, акценты | выберите цвет |
| `--color-link` | Ссылки | как accent или отдельно |
| `--color-border` | Рамки, разделители | `#e0e0e0` |

## Шрифты

| Название | Назначение | Пример |
|----------|------------|--------|
| `--font-primary` | Основной текст | `"PT Sans", sans-serif` или системный стек |
| `--font-heading` | Заголовки | тот же или другой |

## Размеры и отступы

| Название | Назначение | Пример |
|----------|------------|--------|
| `--space-xs` | Минимальный отступ | `4px` |
| `--space-s` | Маленький | `8px` |
| `--space-m` | Средний | `16px` |
| `--space-l` | Большой | `24px` |
| `--space-xl` | Секции | `32px` / `48px` |
| `--max-width` | Макс. ширина контента | `1200px` |
| `--radius` | Скругление кнопок/карточек | `8px` |

## В CSS

В `site/css/` можно завести файл `tokens.css` и перенести туда переменные:

```css
:root {
  --color-bg: #ffffff;
  --color-text: #1a1a1a;
  --color-accent: #2e7d32;
  --font-primary: "PT Sans", sans-serif;
  --space-m: 16px;
  --space-l: 24px;
  --max-width: 1200px;
  --radius: 8px;
}
```

Дальше в вёрстке использовать: `color: var(--color-text);`, `margin-bottom: var(--space-l);` и т.д.
