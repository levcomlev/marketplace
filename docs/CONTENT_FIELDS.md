# Поля карточки товара (product.json)

Только основное: тип, предмет, формат, страна, материал, описание. Инфографику — отдельно позже.

## Основные

| Поле | Тип | Описание |
|------|-----|----------|
| id | строка | Идентификатор, совпадает с именем папки |
| title | строка | Название товара |
| description_short | строка | Краткое описание для карточки в каталоге |
| description_full | строка | Полное описание для страницы товара |

## Характеристики

| Поле | Тип | Примеры значений |
|------|-----|-------------------|
| type | строка | «Обучающий плакат», «Обучающие карточки», «Планинг» |
| subject | строка | «Музыка», «Информатика», «Дисциплина и планирование», «Развивающие для детей», «Йога» |
| format | строка | «A1», «A4», «A3» |
| country | строка | «Россия» |
| material | строка | «Бумага» |

## Ссылки и цифровая версия

| Поле | Тип |
|------|-----|
| link_ozon | строка, URL или пусто |
| link_wb | строка, URL или пусто |
| has_digital | boolean |
| digital_price | число или null |
| digital_format | строка или null (например "PDF") |

## Медиа

| Поле | Тип |
|------|-----|
| image_main | путь к основному фото (используется в галерее, если нет images) |
| image_thumb | путь к превью для каталога (сетка на главной) |
| images | массив строк, опционально | Пути к фото для галереи. Можно не задавать: если в `site/images/products/<id>/` лежат файлы **1.jpg**, **2.jpg**, **3.jpg** и т.д., скрипт сборки подхватит их сам; **1** считается главным фото. |

## Поля для карточки (добавим позже)

Для блока «рейтинг + отзывы» и блока «цена» на карточке (когда решим показывать):

| Поле | Тип | Описание |
|------|-----|----------|
| rating | число или null | Рейтинг, напр. 4.9 |
| reviews_count | число или null | Количество отзывов |
| price | число или null | Текущая цена (если ведём на сайте) |
| price_old | число или null | Старая цена для скидки |
| discount_percent | число или null | Процент скидки, напр. 89 |

Карточка в вёрстке: блоки показываются только если данные есть (один компонент, опциональные блоки).

---

При вёрстке: блок «Характеристики» — type, subject, format, country, material. Блок «Описание» — description_full.
