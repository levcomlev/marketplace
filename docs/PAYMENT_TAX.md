# Оплата и налоговая для ИП

Чтобы вас не беспокоила налоговая при приёме оплаты на сайте, нужно соблюдать 54-ФЗ: **каждая оплата должна сопровождаться чеком** (онлайн-касса или передача данных в ОФД).

## Что делать

Выбрать платёжную систему, которая **сама формирует и отправляет чеки** в ФНС (или в сервис фискализации). Тогда вам не нужно вручную заниматься кассой.

## Варианты для ИП (кратко)

| Сервис | Чеки по 54-ФЗ | Комментарий |
|--------|----------------|--------------|
| **ЮKassa** (Яндекс) | Да, через партнёров (например, «Айтида», «Пэйментэкс», «Модулькасса») или свою касса | Популярный вариант, много инструкций. Нужно подключить одну из интеграций для чеков. |
| **Тинькофф** (Эквайринг / ОПЦ) | Да, встроенная фискализация | Удобно, если уже работаете с Тинькофф. |
| **Robokassa** | Да, через партнёров | Также распространён. |
| **CloudPayments** | Да | Есть фискализация для РФ. |

При подключении указываете, что нужна **передача данных для чеков** (54-ФЗ). Сервис будет отправлять чеки за вас; в личном кабинете ФНС они будут видны.

## Что уточнить при выборе

1. Поддержка **ИП** (не только ООО).
2. **Автоматическая отправка чеков** при оплате (без ваших ручных действий).
3. Совместимость с вашей **системой налогообложения** (УСН, патент и т.д.) — обычно все поддерживают.

## Итог

Оформили ИП — этого достаточно. Дальше: договор с платёжкой + включённая фискализация (чеки). Тогда налоговая получает данные о платежах автоматически, и отдельно вас «беспокоить» по чекам не будет. Детали подключения смотрите на сайтах ЮKassa, Тинькофф или выбранного сервиса.
