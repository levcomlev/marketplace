# Структура сайта: страницы и блоки для вёрстки

Список страниц и блоков, чтобы верстать по шагам и не переделывать всё с нуля.

## Страницы

| № | Страница | Назначение |
|---|----------|------------|
| 1 | Главная (`index.html`) | О проекте, хиты/категории, призыв «Выберите, где купить» |
| 2 | Каталог | Список карточек товаров (плакатов) |
| 3 | Карточка товара | Один плакат: фото, описание, кнопки Озон / WB / Цифровая версия |
| 4 | Контакты (или футер) | Как связаться, ссылки на маркетплейсы |
| 5 | (Позже) Страница оплаты / успеха | После покупки цифровой версии |

Лендинги под рекламу (отдельные страницы под категории) можно добавить по тому же шаблону, не трогая уже готовые.

## Блоки для главной

- Шапка: логотип/название, навигация (Главная, Каталог, Контакты).
- Герой: заголовок, короткий текст, призыв в каталог.
- Блок «Где купить»: Озон, Wildberries, Цифровая версия на сайте (иконки/кнопки).
- Хиты или категории плакатов (карточки со ссылками в каталог).
- Подвал: контакты, ссылки на Озон/WB, политика конфиденциальности (позже).

## Блоки для каталога

- Та же шапка.
- Заголовок страницы.
- Сетка карточек товаров (фото, название, краткое описание, кнопки Озон / WB / Цифровая версия).
- Подвал.

## Блоки для карточки товара

- Шапка.
- Фото плаката, название, полное описание.
- Кнопки: «Купить на Озоне», «Купить на Wildberries», «Купить цифровую версию» (цена на сайте).
- Подвал.

Вёрстку удобно делать в порядке: главная → каталог → карточка товара. Каждый шаг — отдельный коммит.
